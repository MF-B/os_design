# 缓冲区系统设计与实现
这是一个模拟生产者-消费者模型的缓冲区系统，该项目演示了操作系统中的基础概念，如缓冲区管理和生产者-消费者问题的解决方案。

## 项目介绍
本项目实现了三个核心组件：

- Buffer: 实现了一个带容量限制的字符缓冲区
- Producer: 按照指定频率向缓冲区写入随机字符
- Consumer: 可以从缓冲区读取字符或将字符从一个缓冲区移动到另一个缓冲区

这个系统可以用于模拟多种实际场景，例如：

- 操作系统中的进程间通信
- 网络数据包的缓冲和处理
- 多线程应用中的数据交换

## Usage
运行主程序
```bash
make
# 或
make run
```

运行测试
```bash
make test
```

清理项目
```
make clean
```
